# AutSpark Nordic Presentation

Professional Reveal.js presentation for AutSpark's Nordic market pitch.

## Features

- 12 slides covering the full AutSpark value proposition
- Reveal.js framework for smooth transitions
- Presenter notes support (press 's' during presentation)
- Mobile responsive
- Arrow key navigation
- Print to PDF support

## Local Development

```bash
npm install
npm start
```

Visit: http://localhost:3000

## Railway Deployment

1. Push this directory to a git repository
2. Connect to Railway
3. Railway will auto-detect the Dockerfile
4. Deploy!

## Presentation Controls

- **Arrow Keys**: Navigate slides
- **Space**: Next slide
- **'s' Key**: Open speaker notes
- **'f' Key**: Fullscreen
- **Esc**: Slide overview
- **'?'**: Show all keyboard shortcuts

## Customization

Edit `/public/index.html` to update slides content or styling.

## Export to PDF

1. Open presentation in Chrome
2. Add `?print-pdf` to URL
3. Print to PDF with landscape orientation
